# 🛒 Clone do Mercado Livre

Projeto de **clone da interface do Mercado Livre**, desenvolvido com **HTML, CSS e JavaScript**, com o objetivo de praticar desenvolvimento **Front-End** e recriar a estrutura visual de um grande e-commerce.

A aplicação simula a experiência de navegação em uma página de produtos, mostrando itens, imagens e interações básicas.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## ⚙️ Funcionalidades

* 🛍️ Interface inspirada no Mercado Livre
* 📦 Exibição de produtos
* 🖼️ Imagens dos produtos
* 🎨 Layout estilizado com CSS
* ⚡ Interações com JavaScript

---

## 📂 Estrutura do projeto

```
clone-mercado-livre
│
├── index.html
├── styles.css
├── script.js
```

---

## ▶️ Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/seu-usuario/clone-mercado-livre.git
```

2. Entre na pasta do projeto

```
cd clone-mercado-livre
```

3. Abra o arquivo

```
index.html
```

no navegador.
